## چرا این پروژه؟

هنرستان عصر صنعت به وب‌سایتی نیاز داشت که هویت آن را منعکس کند — یک مدرسه فنی و حرفه‌ای پیشرو برای پسران در فردیس، البرز، ایران. این پروژه یک تجربه وب دوزبانه (فارسی/انگلیسی) آماده تولید ارائه می‌دهد:

- **بدون ابزار بیلد** — بدون npm، بدون webpack، بدون فریم‌ورک. فقط `index.html` را باز کنید.
- **پشتیبانی کامل RTL** — چیدمان راست به چپ با فونت وزیرمتن فارسی.
- **طراحی موبایل‌اول** — عملکرد بی‌نقص در گوشی‌ها، تبلت‌ها و دسکتاپ‌ها.
- **دسترس‌پذیر** — برچسب‌های ARIA، ناوبری با کیبورد، HTML معنایی.
- **آماده سئو** — Open Graph، Twitter Cards و JSON-LD داده‌های ساختاریافته.

## شروع کار

این یک وب‌سایت استاتیک است. نیازی به مراحل بیلد نیست.

### گزینه ۱: باز کردن مستقیم

فایل `index.html` را در هر مرورگر مدرن باز کنید.

### گزینه ۲: سرور محلی

```bash
# پایتون
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

سپس به `http://localhost:8000` مراجعه کنید.

### گزینه ۳: Live Server در VS Code

افزونه **Live Server** را نصب کنید، روی `index.html` کلیک راست کنید و **Open with Live Server** را انتخاب کنید.

## استقرار

این سایت را می‌توان در هر مکانی که فایل‌های استاتیک را سرو می‌کند مستقر کرد:

- **GitHub Pages** — به `main` پوش کنید، Pages را در تنظیمات ریپو فعال کنید
- **Netlify** — درگ اند دراپ یا اتصال به ریپو
- **Vercel** — وارد کردن پروژه، استقرار بدون تنظیم
- **هر وب سروری** — آپلود فایل‌ها از طریق FTP/SFTP
